# guess-number

## Back-end

Open a Terminal and run:

```sh
cd back-end
npm install
```

### Usage

Start the app by typing:

```sh
npm start
```

## Front-end

Open a Terminal and run:

```sh
cd front-end
npm install
```

### Usage

Start the server by typing:

```sh
npm start
```

### Browser

In your browser go to:

```sh
http://localhost:3000/
```

### Testing

Start testing by typing:

```sh
npm test
```
